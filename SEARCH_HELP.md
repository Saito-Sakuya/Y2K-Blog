# 搜索指南

Y2K Pixel Blog 的搜索系统支持多种高级语法，可以精准定位内容。

---

## 基础搜索

直接输入关键词，将在**标题**、**标签**和**摘要**中匹配结果。

```
Y2K
```

---

## 标签搜索 `#`

以 `#` 开头加标签名，精确匹配标签。

```
#科幻
#视觉小说
```

点击文章页面上的任意标签，会自动触发 `#标签名` 搜索。

---

## 全文搜索 `~`

以 `~` 开头，在文章**正文内容**中进行全文搜索。

```
~殖民网络
~星际旅行
```

适合查找记得正文中提到过、但不确定标题的内容。

---

## 板块浏览 `@`

以 `@` 开头加板块 slug（板块的英文标识），浏览该板块下的所有内容。

```
@start
@tech-notes
```

---

## 组合搜索 `@板块 + 条件`

在 `@板块` 后加空格，再接搜索条件，缩小搜索范围到特定板块内。

### 板块内搜索标题

```
@start 星之彼方
```

### 板块内标签筛选

```
@start #科幻
```

### 板块内全文搜索

```
@start ~像素风格
```

---

## 语法速查表

| 语法 | 示例 | 说明 |
| :--- | :--- | :--- |
| `关键词` | `Y2K` | 匹配标题 + 标签 + 摘要 |
| `#标签` | `#科幻` | 精确匹配标签 |
| `~文本` | `~殖民网络` | 全文正文搜索 |
| `@板块` | `@start` | 浏览板块全部内容 |
| `@板块 关键词` | `@start 星之彼方` | 板块内搜索标题 |
| `@板块 #标签` | `@start #科幻` | 板块内标签筛选 |
| `@板块 ~文本` | `@start ~像素` | 板块内全文搜索 |

---

## 标签云

点击搜索框时可以看到所有可用标签的列表，点击任意标签即刻开始搜索。标签按使用次数排列，频率越高的标签排在越前面。

---

## 备注

- 搜索不区分大小写
- 板块 slug 可以在菜单中查看（鼠标悬停在板块名称上）
- 搜索结果按日期倒序排列，最新内容优先
- Rating 类型的搜索结果会附带评分
